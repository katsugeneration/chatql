# ChatQL
GraphQL based chat engine

## Dependencies
This module depend following system environment.

- python >= 3.6
- mongodb >= 4.0

Following dependency python modules are installed when `pip install`.

- graphene >= 2.1
- mongoengine >= 0.16.3
